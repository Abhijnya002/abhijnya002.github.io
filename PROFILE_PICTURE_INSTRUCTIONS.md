# Profile Picture Location for About Me Section

## File Location:
Place your profile picture in the **`/public/`** folder (same level as `index.html`)

## Required Filename:
**`Abhijnya-Profile-Photo.webp`**

## Full Path:
```
/public/Abhijnya-Profile-Photo.webp
```

## Image Format:
- **Recommended**: `.webp` format (best quality/size ratio)
- **Alternative**: `.jpg`, `.png` (if using these, update the filename in AboutPage.js)

## Steps:
1. Save your profile picture as `Abhijnya-Profile-Photo.webp`
2. Place it in the `/public/` folder
3. If using a different format (`.jpg` or `.png`), update line 94 in `src/components/AboutPage/AboutPage.js`:
   - Change: `Abhijnya-Profile-Photo.webp`
   - To: `Abhijnya-Profile-Photo.jpg` (or `.png`)
